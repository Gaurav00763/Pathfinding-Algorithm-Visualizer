# Pathfinding-Algorithm-Visualizer

An interactive web-based application to visualize and compare popular pathfinding algorithms like BFS, DFS, A*, and Hill Climbing in real time.

---

## 🌟 Features
- 📍 Interactive grid system (start, end, walls)
- ⚡ Real-time algorithm visualization
- 🧠 AI-based A* heuristic search
- 📊 Performance metrics (explored nodes, path length)
- 🎛️ Speed control & grid customization
- 🔄 Multiple algorithms (BFS, DFS, A*, Hill Climbing)
